- ✅ **Рендеринг узлов в viewport** - все узлы отображаются с правильными стилями

## 🚀 Как использовать

### Запуск
```bash
cd refactored
npm run dev
```

Откройте http://localhost:3000

### Горячие клавиши
- **Delete** - удалить выбранный узел
- **F2** - переименовать узел
- **Ctrl+D** - дублировать узел
- **Arrow Up/Down** - навигация по дереву
- **Escape** - снять выделение

### Функции
- **Drag & Drop** - перетаскивайте узлы в дереве
- **Right Click** - контекстное меню
- **Double Click** - переименование узла
- **Properties Panel** - редактируйте свойства в реальном времени
- **Add Node** - кнопка "+" в панели Scene
